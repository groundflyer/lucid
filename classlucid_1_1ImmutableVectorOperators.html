Detailed Description

template<typename T, std::size_t N, template< typename, std::size_t > typename Container, template< typename, std::size_t, template< typename, std::size_t > typename > typename VectorType>
class lucid::ImmutableVectorOperators< T, N, Container, VectorType >

Meta-class defining common vector operators that don't modify values in-place.

Note
Comparison operators defined here are numpy-like: they perform comparison on all the elements returning vector of bool instead of commonly used lexicographic order.
